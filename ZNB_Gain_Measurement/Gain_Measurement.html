Project Overview

Calibration and Gain Measurement of Amplifier

Using RsInstrument, an open-source Python module, this project facilitates remote-control communication with Rohde & Schwarz SCPI-based Test and Measurement Instruments.

Software Package

RsInstrument is a Python module designed for remote control of Rohde & Schwarz test and measurement devices. It's available under the MIT License.
